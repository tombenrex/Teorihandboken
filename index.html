HTML & CSS

Html

Ett HTML dokument består av olika taggar som ger plats för att skapa innehåll på sidan. Här placerar vi olika element med texter, rubriker, bilder, knappar, formulär för inputs där vi kan skriva in text och mycket mer.

CSS

CSS uppgift är att koppla ihop med HTML elementen för att bestämma vilken färg, storlek, placering de i sin tur ska ha. Användningsområdet är stort men för att beskriva det kort, bestämmer det vilket utseende eller vilka animationer vi vill ha.

Layouttekniker CSS

CSS Framework

Här importeras kod från färdiga bibliotek, ett nytt sätt att skapa. En stor fördel är att man sparar tid, eftersom det är mer effektivt än att skapa från grunden. Nackdel är att man kanske inte använder sig av all kod, vilket i sin tur skapar större filer som innebär längre laddningstider.

Css float property

Något som användes mer förr. Detta fungerar med alla äldre webbläsare och kräver inte lika mycket kod. Nackdelen är det är svårare att få till en bättre konstruktion och centrera innehållet

Css flexbox

En modern teknik som används idag av utvecklare. Det har fördelar som att det både är lätt att förstå och använda. Fördelaktigt när du vill centrera innehållet eller placera det i en rad.

Css grid

Även denna är en modern teknik som gör det till en fördel om du vill skapa helskärms layouter. En mer avancerad metod där man inte behöver lika mycket html kod för att skapa extra elements som exempelvis flexbox. Med detta kan vi placera element i specifika ”grid-areas”.

javascript

Stort användningsområde

Några exempel på vad man kan gör med JS:

  • Hantera användarens input (musklick, tangenttryck, scroll) och koppla dem till funktioner.
  • Validera formulär och beräkna datan.
  • Hämta olika API, importera dessa, skapa i tabeller.
  • Manipulera och byta ut eller skapa nya element i HTML/CSS.
  • Skapa effekter.
